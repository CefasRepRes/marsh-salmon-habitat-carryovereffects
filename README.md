> Marsh, J. E., Gillson, J. P., Riley, W. D., Elliott, S. A. M, Gregory, S. D. (2026) Understanding immediate and carry-over effects to prioritise habitat protection and restoration: a case study of threatened Atlantic salmon. *Biological Conservation*, Volume 317, 111822. doi: https://doi.org/10.1016/j.biocon.2026.111822

## Simulations & code

This is code to simulate and fit to our model to (simulated) data and present the results.

A compiled html showing the codes and plots is available in [marsh-salmon-habitat-carryovereffects-simulation.html](marsh-salmon-habitat-carryovereffects-simulation.html).

To compile it yourself, you should run the file through [quarto](https://quarto.org/). For instructions on doing that, see [https://quarto.org/docs/get-started/hello/rstudio.html](https://quarto.org/docs/get-started/hello/rstudio.html).

## Data availability

This document uses simulation data. The data used in the analysis are available at Zenodo from [https://doi.org/10.5281/zenodo.19205913](10.5281/zenodo.19205913).
